# 📄 PRD — BizPage AI
> **소상공인을 위한 AI 가게소개페이지 자동 생성 솔루션**
> Gemini 3 서울 해커톤 | For Good 트랙

---

## 1. 제품 비전 (Product Vision)

> "사장님은 요리만 하세요. 나머지는 BizPage AI가 합니다."

마케팅 전문성이 없는 소상공인도 사진 몇 장과 기본 정보만으로 전문가급 가게소개 페이지를 즉시 만들 수 있도록 돕는 AI 서비스입니다.

---

## 2. 문제 정의 (Problem Statement)

- 한국 소상공인의 **40%가 창업 후 3년 내 폐업**하며, 폐업 원인 1위는 **매출 부진(86.7%)** 입니다.
- 대부분의 소상공인은 마케팅 전문성이 없고, 마케팅 대행사 비용은 월 수백만 원 수준입니다.
- 네이버·카카오 등록은 복잡하고, 직접 홍보 페이지를 만드는 것은 기술적으로 어렵습니다.
- 사장님은 요리·서빙·재고관리만으로도 이미 시간이 부족합니다.

**핵심 사용자:**
> 동네 음식점·카페·소매점 사장님 — 디지털 전문성 없이도 가게를 알리고 싶은 분

---

## 3. 핵심 기능 (Core Features)

### Step 1 — AI 비서 (AI Chat) — 대화형 온보딩
- 서비스 최초 진입 시 AI 비서와 대화로 시작
- Gemini가 사장님에게 필요한 정보를 자연스럽게 질문하며 수집 (상호명, 가게 특징, 분위기 등)
- 대화가 끝나면 입력 폼을 거치지 않고 바로 페이지 생성으로 연결
- "우리 가게 랜딩페이지 만들어줘" 한 마디로 전체 플로우 시작 가능
- 코딩·디자인 지식 없는 사장님을 위한 가장 자연스러운 진입점

### Step 2 — 가게소개페이지 생성기
- **입력 항목 (최소화):**
  - 상호명 (가게 이름)
  - 가게 소개 (인삿말)
  - 가게 주소
  - 대표 사진 업로드
- Gemini가 업로드된 사진을 멀티모달로 분석 → 가게 분위기·메뉴 특징 자동 태깅
- "마법처럼 랜딩페이지 만들기" 버튼 한 번으로 전문가급 HTML 페이지 즉시 생성

### Step 3 — 고객 의도(Intent) 기반 지능형 최적화

**고정 영역 (Fixed Zone)**
- 브랜드 신뢰도를 유지하는 기본 정보: 상호명, 주소, 구글 맵 연동

**가변 영역 (Dynamic SEO)**
- 유입 경로(UTM 파라미터)에 따라 히어로 섹션의 카피와 이미지를 Gemini가 자동 맞춤 설정
  - 예: '회식 장소' 검색 유입 → 단체석 인프라 강조 카피 + 넓은 홀 이미지
  - 예: '데이트 코스' 검색 유입 → 감성적인 비주얼 중심 구성 + 분위기 강조 카피

### Step 4 — 전문가 검토 및 승인 프로세스 (Approval Loop)

- **기획 리포트:** AI가 페이지를 변경하기 전, 사장님께 '기획 의도'와 '변경 전/후 미리보기'를 먼저 제공
- **최종 확정:** 사장님이 승인한 내용만 프로덕션 환경에 반영 → 비즈니스 통제권 유지
- AI가 자동으로 바꾸는 것이 아니라, **사장님이 결정권을 갖는 구조**

### Step 5 — 가게소개페이지 관리 (대시보드)

- 생성된 페이지를 목록으로 관리
- 실시간 미리보기 제공
- HTML 파일 다운로드 기능
- 페이지 이름: 자동 타임스탬프로 저장 (예: `가게소개페이지-26.02.28 12:54.html`)

## 4. 생성 페이지 구조 (Landing Page Architecture)

```
[상단] 히어로 섹션
  - 상호명 텍스트 오버레이
  - 업로드된 대표 사진 (고해상도 비주얼)

[중단] 비즈니스 정보 카드
  - 사장님 인삿말
  - 가게 주소 정보

[하단] 로컬 인터랙션
  - "구글 맵에서 위치 보기" 버튼
  - 실제 방문 유도
```

## 5. 기술 스택 (Tech Stack)

| 영역 | 기술 |
|------|------|
| 프론트엔드 | Next.js |
| AI 모델 | Gemini 3 Flash (멀티모달 이미지 분석 + 텍스트 생성) |
| 페이지 생성 | HTML / Tailwind CSS 자동 빌드 |
| 배포 | GCP Cloud Run |
| 스토리지 | Firestore (페이지 저장 및 관리) |
